# Οδηγίες Εκτέλεσης Cinema Django Backend

## Απαιτήσεις
- Python 3.8+
- pip
- Virtual Environment

## Βήματα Εκτέλεσης

### 1. Εγκατάσταση Dependencies
```bash
pip install -r requirements.txt
```

### 2. Εφαρμογή Μεταναστεύσεων (Migrations)
```bash
python manage.py migrate
```

### 3. Δημιουργία Superuser (προαιρετικό για admin)
```bash
python manage.py createsuperuser
```
Ακολουθήστε τις οδηγίες για username, email, password.

### 4. Εκτέλεση του Server
```bash
python manage.py runserver
```
Ο server θα τρέξει στο http://127.0.0.1:8000/

### 5. Πρόσβαση στο Admin Panel
Μετά τη δημιουργία superuser, πηγαίνετε στο http://127.0.0.1:8000/admin/

## API Endpoints
- Movies: http://127.0.0.1:8000/api/movies/
- Screenings: http://127.0.0.1:8000/api/screenings/
- Bookings: http://127.0.0.1:8000/api/bookings/

## Αντιμετώπιση Προβλημάτων
- Αν δείτε σφάλματα με migrations, βεβαιωθείτε ότι έχετε εφαρμόσει όλες τις μεταναστεύσεις.
- Αν το frontend δεν μπορεί να συνδεθεί, βεβαιωθείτε ότι ο backend τρέχει στο σωστό port.
