# Οδηγίες Εκτέλεσης - Running Instructions

## Απλά βήματα για να τρέξετε το Cinema Django Backend

### 1. Ενεργοποιήστε το Virtual Environment
```
C:\Users\jimpa\PycharmProjects\Cinema-Django-Backend\.venv\Scripts\activate
```

### 2. Εφαρμόστε τις μεταναστεύσεις (αν δεν έχουν γίνει ήδη)
```
python manage.py migrate
```

### 3. Δημιουργήστε έναν superuser για το admin (προαιρετικό, αλλά χρήσιμο)
```
python manage.py createsuperuser
```
Ακολουθήστε τις οδηγίες για username, email και password.

### 4. Τρέξτε τον server
```
python manage.py runserver 8000
```

### 5. Πρόσβαση στην εφαρμογή
- **Κεντρική σελίδα:** http://127.0.0.1:8000/
- **Admin panel:** http://127.0.0.1:8000/admin/
- **API:** http://127.0.0.1:8000/api/

### Σημείωση
Το project είναι έτοιμο και λειτουργικό. Αν έχετε προβλήματα, βεβαιωθείτε ότι έχετε εγκαταστήσει τις απαιτούμενες βιβλιοθήκες από το `requirements.txt`.
